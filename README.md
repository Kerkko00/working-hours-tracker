# Working Hours Tracker

Working Hours Tracker is an Android application designed to help you manage your professional working hours directly from your mobile device. It provides fast and easy way to keep track of overtime with customizable alert thresholds. The application removes the need to input any extra information such as project names or customer information making it ideal for lightweight working hour and overtime tracking.

## Features

- **Easy Time Entry**: Quickly add your daily working hours using a stylized input form with calendar integration.
- **Smart Alerts**: 
    - Set custom weekly working hour thresholds (e.g. 37,5 hours).
    - Tracks ongoing and configurable amount of past weeks for exceeding working hours.
    - Get visual alerts on the Home screen when set limits are exceeded.
- **Data Persistence**: Your data is saved locally in structured JSON format focusing on data privacy.
- **Modern UI**: Clean Material 3 design with adaptive navigation and great performance.

## Installation for Developers

To run this project locally, you will need:
- **Android Studio** (Ladybug or newer recommended)
- **Android SDK** (API 34+ recommended)
- **Kotlin 2.0.0+**

### Steps:
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Kerkko00/working-hours-tracker.git
   ```
2. **Open in Android Studio**:
   - Launch Android Studio.
   - Open the cloned project.
3. **Sync Project**:
   - Wait for the Gradle sync to complete. Android Studio will automatically download the necessary dependencies (Compose, Material 3, GSON, etc.).
4. **Run the App**:
   - Connect an Android device via USB or start an Emulator.
   - Click the **Run** button (green arrow) in the toolbar.
